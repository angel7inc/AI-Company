# ブランドQAチェックリスト: Tomo_Angel7

投稿・コンテンツの最終確認前に使用します。基準の根拠は[brand-brief.md](./brand-brief.md)です(内容はここに複製せず、判断基準のみをチェック項目化しています)。

## 共通チェック項目

- [ ] ブランド世界観と一致している
- [ ] ブランド人格を崩していない
- [ ] AIらしい文章になっていない
- [ ] CTAが自然
- [ ] 保存したくなる内容
- [ ] シェアしたくなる内容
- [ ] ブランド価値を高める内容
- [ ] 法律・著作権・薬機法等の問題なし
- [ ] SEO/Instagramアルゴリズムに配慮
- [ ] 誤字脱字がない

## ブランド固有のチェック項目
| 観点 | 基準 |
|---|---|
| 断定表現 | 「絶対」「必ず」等の断定的な未来予言をしていないか |
| 依存誘導 | 依存を促す表現になっていないか |
| 恐怖煽り | 不安を過度に煽っていないか |
| スピリチュアル過多 | 専門用語・霊的表現を使いすぎていないか |
| 上から目線 | 説教・断定・見下すような言い回しになっていないか |
| LINE誘導の自然さ | LINE導線(brand-brief 19章)の頻度・トーンから外れていないか |
| 独自性 | 既存投稿・競合との重複が少ないか(`research-log.md`で確認) |

チェック項目は、同じブランドフォルダ内の [brand-brief.md](./brand-brief.md) の方針をもとに具体化しています。brand-brief更新時は、このチェックリストとの整合も見直してください。

## メディア固有のブランド基準(画像・動画)
`shared-image-quality-checker`・`shared-video-quality-checker`から`shared-brand-quality-checker`へブランド判定が委譲された際、メディア種別ごとに使う基準です(技術基準そのものは各Quality Checkerが担当し、ここはブランド適合の観点のみ)。

| メディア | 観点 | 基準 |
|---|---|---|
| 画像 | 世界観の色調 | 淡い色調・月/星/光モチーフ(brand-brief 12章)から外れていないか |
| 画像 | テンプレート整合 | `ig-creative-team`のテンプレートを使う場合、ブランドカラー・フォントが正しく適用されているか |
| 動画 | トーン・テンポ | 落ち着いた・寄り添うトーン(brand-brief 11章)に沿っているか。過度に煽るテンポになっていないか |
| 動画 | BGM・演出 | 恐怖・不安を煽るBGM/演出(brand-brief 16章のNG表現)になっていないか |
