# レポート

週次・月次・四半期のレポート、およびCEO承認ログをアーカイブする場所です。ファイル名は [命名規則](../../docs/naming-conventions.md) の `{YYYYMMDD}-{事業}-{内容}-{版}` に従います。

## 内容
- `management-kpi`・`management-resource`・`management-quality`が作成する週次/月次スナップショット
- `_shared/sop/management/ceo-approval-flow.md`に基づく承認ログ
- `_shared/sop/management/coo-approval-flow.md`に基づく軽微修正の記録(月次でまとめて記載)
