# SEO(Search & SEO Division)

## この事業について
- 目的: Google検索・WordPressを通じた集客をInstagramに次ぐ第二の柱とする
- 主なターゲット: `_company/brands/tomo-angel7/brand-brief.md`のターゲットペルソナと同一(検索経由の流入者)
- 収益モデル: Instagram投稿 → 保存数分析 → SEO記事化 → WordPress公開 → Google検索流入 → LINE登録 → 鑑定(→スクール・電話占いは将来接続)

## 現在稼働中のエージェント
| エージェントID | 役割 | ステータス |

| `seo-wordpress-content` | SEO記事設計・構成・メタ要素 | draft |
| `seo-technical` | テクニカルSEO | draft |

| `seo-search-console` | 検索パフォーマンス分析 | draft |
| `seo-quality-checker` | SEO記事の100点満点品質チェック | draft |
| `seo-editorial-image` | 記事ごとのAI画像生成(Creative Division) | draft |
