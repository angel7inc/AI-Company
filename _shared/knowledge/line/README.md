## 1. 何を保存する場所か

## 2. 保存してはいけない情報

## 3. 誰が参照するか
将来のLINE担当エージェント。

## 4. 更新ルール
LINE公式アカウントの仕様変更を確認したら更新する。

## 5. 推奨する情報源(Sources)
`official`, `documentation`

## 7. 機密情報の扱い
