# Knowledge: video-production

## 1. 何を保存する場所か
動画編集・構成の一般理論(構成・テンポ・カット割り・BGM・テロップ・冒頭3秒の作り方・視聴維持率を高める型)。ツール非依存の編集理論。AIへのプロンプトの話は`video-generation/`を参照。

## 2. カテゴリの境界線
`video-generation/`のREADMEを参照(同じ境界線表を掲載)。

## 3. 保存してはいけない情報
特定ブランドの動画の実際の構成台本(→`businesses/{事業}/content/`)、実際の視聴維持率等の実績データ(→`businesses/{事業}/data/`)。

## 4. 誰が参照するか
`shared-video-creative-team`、`shared-video-quality-checker`。

## 5. 更新ルール
プラットフォームのアルゴリズム変化(視聴維持率の評価基準等)に応じて更新する。

## 6. 推奨する情報源(Sources)
`book`, `industry-report`, `youtube` ― 具体例: 動画編集の専門書、プラットフォーム公式のクリエイター向け情報

## 7. 関連Knowledge
- `video-generation/`
- `youtube/`
- `instagram/`

## 8. 機密情報の扱い
一般理論は`public`/`internal`。

## 9. 重複防止ルール
同じ編集技法の重複記事を作らず、既存記事へ追記する。
