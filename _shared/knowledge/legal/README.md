# Knowledge: legal

## 1. 何を保存する場所か
薬機法・景品表示法・著作権など、コンテンツ制作に関わる法律の一般知識・注意点。

## 2. 保存してはいけない情報
個別の法律相談内容、顧客の個人情報、専門家未確認の確定していない法的判断。

## 3. 誰が参照するか
`ig-brand-qa`、将来の全事業のQA担当エージェント。

## 4. 更新ルール
法改正があれば速やかに更新し、`last_verified_at`を必ず更新する。専門家に確認した内容は`confidence: expert-reviewed`にする。

## 5. 推奨する情報源(Sources)
`official`, `research-paper` ― 具体例: 消費者庁・厚生労働省等の公式情報、弁護士等専門家の見解

## 6. 関連Knowledge
- `copywriting/`
- `branding/`

## 7. 機密情報の扱い
一般的な法律知識は`public`扱いも可能だが、社内の法的判断・解釈は`internal`以上にする。

## 8. 重複防止ルール
法律分野ごとに1記事を原則とする(例: 薬機法1本、景表法1本)。細分化しすぎない。
