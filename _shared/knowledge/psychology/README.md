# Knowledge: psychology

## 1. 何を保存する場所か
行動心理学・行動経済学・認知バイアス・恋愛心理学など、コンテンツ企画やコピーの根拠となる一般的な心理学・行動経済学知識。CVR改善・UX改善(Conversion Division)の判断根拠もここに含む。

## 2. 保存してはいけない情報
特定個人(顧客)の心理状態の推測・個人情報。

## 3. 誰が参照するか
`ig-content-planner`、`ig-copywriting`、Instagram固有戦略機能(planned / unassigned)。

## 4. 更新ルール
学術的な内容が中心のため更新頻度は低いが、`sources`(出典)を必ず明記する。

## 5. 推奨する情報源(Sources)
`research-paper`, `book` ― 具体例: 学術論文、行動経済学・心理学の専門書

## 6. 関連Knowledge
- `marketing/`
- `copywriting/`

## 7. 機密情報の扱い
一般理論のため`public`/`internal`。個人が特定できる情報は一切含めない。

## 8. 重複防止ルール
同じ心理効果・バイアスについて複数記事を作らず、1つにまとめる。
