# Knowledge: copywriting

## 1. 何を保存する場所か
コピーライティングの型・技法(フック、PASONA、QUEST等の一般的なフレームワーク)。特定ブランドのトーンや実際の文言例は`_company/brands/{ブランドID}/`へ置く。

## 2. 保存してはいけない情報
特定ブランドの実際の投稿文・キャプション(→ブランド層または事業の`content/`)。

## 3. 誰が参照するか
`ig-copywriting`、`ig-content-planner`(および将来、全チャネルのコピー担当エージェント)。

## 4. 更新ルール
実践して効果が高かった型は`confidence`を`verified`へ更新する。

## 5. 推奨する情報源(Sources)
`book`, `industry-report`, `experiment` ― 具体例: コピーライティングの定番書籍、実務での検証結果

## 6. 関連Knowledge
- `psychology/`
- `marketing/`
- `branding/`

## 7. 機密情報の扱い
型・技法は`public`/`internal`。ブランド固有の実例は含めない。

## 8. 重複防止ルール
既存の型と重複する新規記事を作らず、既存記事にバリエーションとして追記する。
