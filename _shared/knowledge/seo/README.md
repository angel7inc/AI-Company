# Knowledge: seo

## 1. 何を保存する場所か
検索エンジン最適化(SEO)の仕組みと知識。将来のYouTube・ブログ等のSEOにも共通して使う。

## 2. 保存してはいけない情報
特定事業の実際の検索順位・アクセス数等の実績データ。

## 3. 誰が参照するか
将来のYouTube/ブログ担当エージェント、`ig-research`(ハッシュタグ・検索性の観点)。

## 4. 更新ルール
検索アルゴリズムの変更を確認次第、既存記事を更新する。

## 5. 推奨する情報源(Sources)
`official`, `documentation`, `industry-report` ― 具体例: Google検索セントラル等の公式情報、信頼できるSEO専門メディア

## 7. 機密情報の扱い
一般知識は`public`/`internal`。自社の検索順位等の実績は含めない。

## 8. 重複防止ルール
同じSEO手法の重複記事を作らず、既存記事へ追記する。
