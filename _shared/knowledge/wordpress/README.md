# Knowledge: wordpress

## 1. 何を保存する場所か
WordPressというツールそのものの使い方・仕様(投稿・固定ページ・カテゴリ/タグ・パーマリンク設定・プラグイン・Schema設定等)。`studio/`・`canva/`と同じ「ツール知識」カテゴリ。

## 2. 保存してはいけない情報
SEO理論そのもの(→`seo/`)、特定サイトの実際の記事内容・アクセス数等の実績データ(→`businesses/seo/data/`)。

## 3. 誰が参照するか
`seo-wordpress-content`、`seo-technical`、`seo-internal-link`、`seo-editorial-image`。

## 4. 更新ルール
WordPress本体・主要プラグインの仕様変更を確認次第、既存記事を更新する。

## 5. 推奨する情報源(Sources)
`official`, `documentation` ― 具体例: WordPress公式ドキュメント、主要SEOプラグインの公式ヘルプ

## 6. 関連Knowledge
- `seo/`
- `claude-code/`

## 7. 機密情報の扱い
一般知識は`public`/`internal`。自社サイトの管理画面URL・認証情報等は含めない。

## 8. 重複防止ルール
同じ機能・設定の重複記事を作らず、既存記事へ追記する。
