# ブランドQAチェックリスト テンプレート

新しいブランドを作るときは、このファイルをコピーして `_company/brands/{ブランドID}/qa-checklist.md` として保存します。各チャネルのBrand QAエージェントは、担当ブランドのこのファイルを参照します。

## 共通チェック項目

- [ ] ブランド世界観と一致している
- [ ] ブランド人格を崩していない
- [ ] AIらしい文章になっていない
- [ ] CTAが自然
- [ ] 保存したくなる内容
- [ ] シェアしたくなる内容
- [ ] ブランド価値を高める内容
- [ ] 法律・著作権・薬機法等の問題なし
- [ ] SEO/Instagramアルゴリズムに配慮
- [ ] 誤字脱字がない
