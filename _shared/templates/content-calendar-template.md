# 投稿カレンダー テンプレート

事業を問わず使う投稿カレンダーの共通フォーマットです。各事業の `content/` フォルダに `calendar.md` としてコピーして使います。

| 日付 | 投稿タイプ | テーマ・ネタ | ステータス | 担当エージェント |
|---|---|---|---|---|
| YYYY-MM-DD | 投稿 / リール / ストーリーズ | | 下書き / レビュー中 / 確定 / 公開済み | |

**ステータスの意味**
- 下書き: エージェントが企画・作成した段階
- レビュー中: COOまたはCEOの確認待ち
- 確定: 公開予定が決まった状態
- 公開済み: 実際に投稿された状態
