# SOP: video

## 何を書く場所か
Instagram Reels・YouTube Shorts・YouTube通常動画・広告動画・LINE動画・LP動画など、動画制作の具体的な実行手順。チャネルごとに分割せず、動画という制作物の種類でカテゴリを1つに集約する。

## 書いてはいけないもの
動画編集・生成の理論そのもの(→`_shared/knowledge/video-production/`・`video-generation/`)、実際の視聴維持率等の実績データ(→各事業の`data/`)、ブランド固有のトーン・NG表現そのもの(→`_company/brands/{ブランドID}/brand-brief.md`)。

## 誰が参照するか
`shared-video-creative-team`、`shared-video-prompt-engineer`、`shared-video-quality-checker`、`shared-thumbnail-team`。

## 更新ルール
使用ツール(動画生成AI、Premiere Pro、CapCut、DaVinci Resolve等)・プラットフォーム仕様が変わったら更新する。
