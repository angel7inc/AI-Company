# SOP: research

## 何を書く場所か
悩み調査・競合調査・トレンド調査・市場分析・リサーチ品質チェック・insight統合など、リサーチ業務の具体的な実行手順(どのツールにどんな手順でアクセスし、何を記録するか)。Intelligence Division(Phase7)の全エージェントがこの1カテゴリを共有する(チャネル・分析対象ごとに細分化しない)。

## 書いてはいけないもの
リサーチ手法そのものの一般理論(→`_shared/knowledge/research-methodology/`等の該当カテゴリ)、実際の調査結果データそのもの(→`businesses/{事業}/content/research-log.md`等)。

## 誰が参照するか
`intelligence-market`、`intelligence-consumer`、`intelligence-trend`、`intelligence-insight-synthesizer`、`intelligence-research-quality-checker`、`shared-competitor-intelligence`、`shared-traffic-intelligence`。
*(旧`ig-research`はPhase7のIntelligence Division新設に伴い正式に退役。今後もInstagram専用のリサーチエージェントは新設しない。)*

## 更新ルール
使用ツールの仕様変更でリサーチ手順が変わったら更新する。

- `instagram/`
