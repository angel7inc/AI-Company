# SOP: publishing

## 何を書く場所か
投稿の最終確認・予約投稿・公開の具体的な実行手順(承認フローを含む)。

## 書いてはいけないもの
実際の投稿内容そのもの、CEO承認前の下書き(→事業の`content/`)。

## 誰が参照するか
COO、公開作業を行う全担当。

## 更新ルール
承認フロー・使用ツール(Meta Business Suite等)の変更時に更新する。

## 関連SOP
- `instagram/`
- `analytics/`

## 重要
公開・予約投稿はCEO承認が必須です(会社の承認ルールに準拠)。このカテゴリのSOPには、必ず承認確認のステップを手順内に明記してください。
