# SOP: seo

## 何を書く場所か
SEO記事制作・キーワード戦略・技術SEO・内部リンク設計・EEAT採点・SEO品質チェックの具体的な実行手順。Instagramにおける`instagram/`と同じ位置づけの、SEO/WordPressチャネル専用カテゴリ。

## 書いてはいけないもの
SEO理論そのもの(→`_shared/knowledge/seo/`)、実際の検索順位・アクセス数等の実績データ(→`businesses/seo/data/`)、ブランド固有のトーン・NG表現そのもの(→`_company/brands/{ブランドID}/brand-brief.md`。SOPは参照するが複製しない)。

## 誰が参照するか
`seo-research`、`seo-keyword-strategy`、`seo-wordpress-content`、`seo-technical`、`seo-internal-link`、`seo-eeat-checker`、`seo-quality-checker`。

## 更新ルール
検索アルゴリズム・WordPress仕様の変更に応じて更新する。実行して不備が見つかったら都度修正し、`version`を上げる。
