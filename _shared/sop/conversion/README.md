# SOP: conversion

## 何を書く場所か
LP改善・導線改善・CTA改善・CVR改善・離脱率改善・UX改善・LINE登録率改善、および**ABテストの実施手順・LINE固有のCVR改善手順**(Revenue Division Conversion Team相当)の具体的な実行手順。特定チャネルに属さない横断カテゴリ(`shared-conversion-optimizer`が担当)。

## 書いてはいけないもの
CVR理論・行動心理学理論そのもの(→`_shared/knowledge/marketing/`・`psychology/`・`analytics/`)、実際のCVR実績データ(→各事業の`data/`)、CRM配信・顧客管理の手順(→`crm/`)。

## 誰が参照するか
`shared-conversion-optimizer`。

## 更新ルール
効果が確認できた改善パターンを追加・更新する。
