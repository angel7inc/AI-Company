# SOP: crm

## 何を書く場所か
LINE配信・ステップ配信・顧客管理・セグメント管理・再購入施策・LTV改善(クロスセル・アップセル・紹介・顧客維持)の具体的な実行手順。全チャネル共通の横断カテゴリ(`revenue-crm-manager`・`revenue-ltv-manager`が担当)。

## 書いてはいけないもの
CRM/LTV理論そのもの(→`_shared/knowledge/crm/`)、実際の顧客データ・配信リスト(→各事業の`data/`)、LP/CTA/ABテストの手順(→`conversion/`)。

## 誰が参照するか
`revenue-crm-manager`、`revenue-ltv-manager`。

## 更新ルール
効果が確認できた配信パターン・リテンション施策を追加・更新する。

## 関連SOP
- `conversion/`
- `analytics/`
- `management/`(売上レビューは`review-cycle.md`の月次レビューに統合)

## 重要
顧客への配信・オファー内容は、対象ブランドの`brand-brief.md`のトーン・NG表現に沿っているか、`shared-brand-quality-checker`のチェックを経てから実行する。
