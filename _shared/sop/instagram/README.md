# SOP: instagram

## 何を書く場所か
Instagram事業に関する具体的な業務手順(投稿企画〜公開までの一連の作業の進め方、全体設計の手順等)。

## 書いてはいけないもの
Instagramというプラットフォーム自体の一般知識(→`_shared/knowledge/instagram/`)、ブランド固有の方針そのもの(→`_company/brands/`。SOPはbrand-briefを参照するが内容を複製しない)。

## 誰が参照するか
`ig-content-planner`、`ig-copywriting`、`ig-brand-qa`、`ig-analytics`。

## 更新ルール
実行して不備が見つかったら都度更新し、`version`とRevision Historyに記録する。

## 補足
このカテゴリはInstagram固有のSOP専用です。将来YouTube・TikTok等のチャネルが加わったときは、ここを流用せず`sop/youtube/`のような新しいチャネル別カテゴリを追加してください([`_shared/sop/README.md`](../README.md)の「カテゴリの拡張方針」を参照)。
