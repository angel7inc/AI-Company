# SOP: brand-qa

## 何を書く場所か
全チャネル共通のブランドQA(トーン・世界観・AI臭・禁止表現・一貫性チェック)の具体的な実行手順。`shared-brand-quality-checker`が担当する横断カテゴリ。

## 書いてはいけないもの
ブランド固有の判断基準そのもの(→`_company/brands/{ブランドID}/qa-checklist.md`。SOPは参照するが複製しない)、QA理論一般(→`_shared/knowledge/branding/`)。

## 誰が参照するか
`shared-brand-quality-checker`、`shared-image-quality-checker`、`shared-video-quality-checker`(ブランド判定部分の委譲元として)、`seo-quality-checker`。

## 更新ルール
チェック観点・フェイルクローズの運用ルールが変わったら更新する。この手順の変更は全チャネルに影響するため、Gate1相当の重要な意思決定として扱う。

## 関連SOP
- `instagram/`
- `seo/`
- `design/`
- `video/`

## 重要
評価に必要な情報(対象ブランドの`qa-checklist.md`等)が揃わない場合は、通過させず差し戻す(フェイルクローズ)。
