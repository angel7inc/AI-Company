# プロンプトテンプレート

事業を問わず使い回せる再利用プロンプト集です。良い結果が出たプロンプトはここに残し、次回から使い回します。

## フォルダ構成
| フォルダ | 内容 |
|---|---|
| `image/` | 画像生成AI用の再利用プロンプト(`shared-image-prompt-engineer`が管理) |
| `video/` | 動画生成AI用の再利用プロンプト(`shared-video-prompt-engineer`が管理) |

ブランド固有の値(カラー・フォント名等)はプロンプト内に直接書かず、`_company/brands/{ブランドID}/brand-brief.md`を参照する形にしてください(重複防止)。
